# 로컬 가격검수 웹서비스

## 실행
```bash
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install --upgrade pip
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
copy .env.example .env
python server.py
```

브라우저:
`http://127.0.0.1:8000`

## 주요 기능
- PDF 업로드 후 권역별 가격 검수
- 데이터 관리 모달에서 기준 엑셀 업로드 / 데이터 표 확인
- 증정품 / 사은품 / 세트상품 제외
- 빨간색만으로 할인 판정하지 않고 `SALE`, `할인`, `%`, `특가` 문구가 있을 때만 할인 처리
- 할인율 명시 시 기준가격 할인 후 비교
- 오류 항목과 할인적용 항목 색상 강조
- OpenAI 키가 있으면 AI 보조 파싱 사용, 없으면 규칙 기반 파싱 사용
